# 🚗 Car Price Prediction using Machine Learning

This project is part of my Data Science Internship at CodeAlpha.

---

## 📌 Project Overview
The objective of this project is to build a machine learning model that predicts the selling price of cars based on various features such as fuel type, transmission, and car age.

This project demonstrates the application of regression techniques in real-world scenarios.

---

## 📂 Dataset
The dataset contains information about different cars and their attributes.

Key features include:
- Car_Name
- Year
- Selling_Price
- Present_Price
- Kms_Driven
- Fuel_Type
- Selling_type
- Transmission
- Owner

---

## ⚙️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 🔍 Key Steps Performed
- Data Cleaning (handling missing values)
- Data Transformation (converting categorical data)
- Feature Engineering (creating Car Age feature)
- Splitting dataset into training and testing sets
- Model Training using Random Forest Regressor
- Model Evaluation using R2 Score and Mean Absolute Error

---

## 📈 Model Performance
- R2 Score: Indicates how well the model predicts car prices
- Mean Absolute Error: Measures prediction error

---

## 🚀 Conclusion
This project successfully demonstrates how machine learning can be used to predict car prices based on historical data.

It highlights the importance of data preprocessing, feature engineering, and model evaluation in building effective predictive models.
